# Security & legal contact

## Reporting a vulnerability

Please report security issues privately via GitHub Security Advisories
("Report a vulnerability" on the Security tab) or email
**security@earthquakes-feed.theshelter.app**. Do not open a public issue for
undisclosed vulnerabilities. We aim to acknowledge within **2 business days**.

## DMCA / copyright

- **Designated agent:** to be registered with the U.S. Copyright Office DMCA
  directory before public launch (`copyright@earthquakes-feed.theshelter.app`).
- Send notices to that address. We honor the full DMCA process, including
  counter-notice put-back for content disabled by mistake or misidentification
  (e.g. facts wrongly claimed as copyrighted).
- For data-removal requests generally, see [TAKEDOWN.md](TAKEDOWN.md).

## Response targets

| Request | Target |
|---|---|
| Acknowledge receipt | ≤ 2 business days |
| Live-feed redaction for a colorable claim | ≤ 2 business days |
| Historical rebuild-and-repin | ≤ 7 days |
| Attribution-only fix | ≤ 3 days |

We promise fast redaction on the surfaces we control and are honest that
third-party-cached residue is eventual/best-effort.
