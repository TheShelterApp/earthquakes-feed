# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal
attacks are not tolerated. Report concerns to
**conduct@earthquakes-feed.theshelter.app**. Maintainers may remove comments,
commits, issues, and contributors that violate this code.
